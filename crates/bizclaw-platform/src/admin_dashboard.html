⚙️ SePay — Nhận thanh toán gói SaaS

SePay dùng để nhận tiền khi khách mua gói SaaS BizClaw. Khi nhận được chuyển khoản đúng mã, hệ thống tự kích hoạt plan cho tenant tương ứng.

1. Đăng nhập SePay Dashboard → my.sepay.vn
2. Vào Webhooks → Click "+ Add webhooks"
3. Cấu hình Webhook:
• URL: Loading...
• Event: In_only (chỉ giao dịch nhận)
• Auth: Api_Key

4. Cấu hình hiển thị VietQR (Cho màn hình đăng ký):
1. Pay2S (Tuỳ chọn) → Tích hợp thêm kênh Pay2S (Cấu hình tương tự SePay)
• URL: Loading...
• Auth: Bearer Token / Api Key

Hoặc đăt qua env var BIZCLAW_PAY2S_API_KEY

Ví dụ mua gói: Khách muốn mua gói Pro cho tenant dalat-xinh:
→ Chuyển khoản 499.000đ với nội dung: BIZCLAW-dalat-xinh
→ Hệ thống tự động kích hoạt gói Pro cho tenant đó.

🏢 Quản lý Tenant

Tenant là đơn vị cơ bản trong BizClaw — mỗi tenant là một AI agent instance độc lập.

Tạo Tenant mới

  1. Vào Tenants → Click "+ New Tenant"
  2. Name: Tên hiển thị (VD: "Sales Bot")
  3. Slug: URL identifier (VD: "sales" → sales.bizclaw.vn)
  4. Provider: OpenAI, Anthropic, Ollama, Gemini, DeepSeek, Groq
  5. Model: gpt-4o-mini, claude-3-haiku, llama3.2, v.v.

Xem chi tiết

Click "Detail" để xem thông tin tenant với 5 tab: Overview, Settings, Agents, Channels, Model.

Thao tác

  • ▶ Start / ⏹ Stop: Bật/tắt tenant
  • 🔄 Restart: Khởi động lại
  • 🔑 Reset Pairing: Tạo mã ghép nối mới
  • ✕ Delete: Xóa tenant vĩnh viễn

🤖 Cấu hình Agent

Mỗi tenant có thể có nhiều agent — mỗi agent có model, system prompt riêng.

Thêm Agent

  1. Vào Tenant Detail → tab Agents

Xóa Agent

  • Mô tả rõ vai trò: "Bạn là trợ lý bán hàng chuyên nghiệp"

  • Set tone: "Luôn thân thiện, chuyên nghiệp"

📱 Thiết lập kênh giao tiếp

BizClaw hỗ trợ 6 kênh giao tiếp. Vào Tenant Detail → tab Channels.

Telegram

  1. Mở @BotFather trên Telegram
  2. Gõ /newbot → Đặt tên → Copy Token
  3. Paste token vào field → Bật toggle → Click 💾 Lưu

Zalo OA

  1. Tạo OA trên oa.zalo.me
  2. Lấy App ID, Secret Key, Access Token
  3. Điền vào form Zalo → Click 💾 Lưu

Discord

  1. Tạo bot trên Discord Developer Portal
  2. Copy Bot Token
  3. Invite bot vào server → Paste token → 💾 Lưu

Webhook

Cấu hình endpoint URL để nhận/gửi messages qua HTTP API.

❓ Câu hỏi thường gặp

Tôi có thể tạo bao nhiêu tenant?

Không giới hạn — tùy thuộc vào tài nguyên server. Mỗi tenant tiêu thụ khoảng 50-100MB RAM.

Dùng model nào tốt nhất?

Cho tốc độ: gpt-4o-mini hoặc Groq. Cho chất lượng: gpt-4o hoặc claude-3-sonnet. Cho miễn phí: Ollama local.

Telegram bot không phản hồi?

  • Kiểm tra token đúng chưa
  • Tenant đang "running" chưa
  • Bot đã được add vào group với quyền admin chưa

Quên mật khẩu admin?

Click "Quên mật khẩu" ở trang login, hoặc nhờ superadmin reset qua trang Users.
