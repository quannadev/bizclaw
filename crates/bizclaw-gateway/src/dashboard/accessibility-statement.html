📜 BizClaw Accessibility Statement

✓ Conforms to WCAG 2.1 Level AA

WCAG 2.1 Level AA

1. Our Commitment

BizClaw is committed to ensuring digital accessibility for people with disabilities. We continually improve the user experience for everyone and apply relevant accessibility standards.

We believe that accessibility is not just a legal requirement but a fundamental aspect of providing excellent service to all our users, regardless of their abilities.

2. Conformance Status

BizClaw Dashboard conforms to WCAG 2.1 Level AA.

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. BizClaw has been designed and developed to meet these guidelines.

3. Technical Specifications

Accessibility of BizClaw is reliant on the following technologies:

  • HTML5 - Semantic markup for structure
  • CSS3 - Styling with accessibility in mind
  • JavaScript (ES6+) - Progressive enhancement
  • ARIA 1.1 - Accessible Rich Internet Applications

These technologies are relied upon for conformance with the accessibility standards used.

4. Accessibility Features

BizClaw includes the following accessibility features:

  • Keyboard Navigation - All interactive elements can be accessed using keyboard
  • Skip Links - Quick navigation to main content
  • Focus Indicators - Clear visual focus states for all interactive elements
  • ARIA Labels - Descriptive labels for screen readers
  • Color Contrast - Text meets WCAG AA contrast requirements (4.5:1 minimum)
  • Responsive Design - Works on various screen sizes and devices
  • Screen Reader Support - Compatible with popular screen readers
  • Reduced Motion - Respects user preferences for reduced motion
  • Text Scaling - Supports browser zoom and text resizing

5. Assessment Approach

BizClaw assessed the accessibility of this website using:

  • axe DevTools - Automated accessibility testing
  • Lighthouse - Performance and accessibility audits
  • Manual keyboard navigation testing
  • Screen reader testing (NVDA, VoiceOver, TalkBack)
  • Color contrast analyzer
  • Responsive design testing across devices

6. Known Issues

We are continuously working to improve accessibility. Known issues include:

  • Some third-party embeds may not meet accessibility standards
  • Complex data visualizations may require additional descriptions

We are committed to addressing these issues in future updates.

7. Accessibility Support

BizClaw is tested and works with:

  • Screen Readers: NVDA (Windows), VoiceOver (macOS/iOS), TalkBack (Android)
  • Browsers: Chrome, Firefox, Safari, Edge (latest versions)
  • Operating Systems: Windows 10+, macOS 12+, iOS 15+, Android 12+
